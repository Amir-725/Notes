# Notes - Простое приложение для заметок

Легкое и стильное приложение для создания заметок для Android, созданное в качестве проекта. Оно позволяет быстро добавлять, редактировать и удалять заметки, и все это обернуто в современный интерфейс в стиле 'glassmorphism' для элегантного, 'стеклянного' вида.
